# Change Log

## 0.1.17

- 恢复 VS Code 原生“代码备注”资源树，并保留原生主题、键盘操作和右键菜单。
- 增加默认、信息、成功、警告、危险五种备注样式和实时预览。
- 备注颜色仅显示在“代码备注”视图，不改变系统资源管理器的文件颜色。
- 增加系统资源管理器与“代码备注”之间的双向定位。
- 修复 Git 忽略目录覆盖备注颜色、快捷键目标错误等问题。
- 简化样式选择器并完善中英文界面。

## 0.1.10

- 首次发布到 Visual Studio Marketplace。
- 支持文件与目录备注、搜索、路径迁移和 `.codebase-notes.json` 共享配置。
- 支持 VS Code、TRAE 与 JetBrains IDE 双端协作。
